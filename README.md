# Procedural Fantasy RPG - Project Overview

This is a web-based Procedural Fantasy RPG built with Vite, Express, and Supabase.

## 🏗 Project Structure

- `src/` - All source code (Frontend & Backend).
- `docs/` - Project documentation, roadmaps, and walkthroughs.
- `public/` - Static assets.
- `index.html` - The main entry point for the frontend.
- `package.json` - Dependencies and scripts.

## 🚀 Key Scripts

- `npm run dev` - Starts the Vite development server for the frontend.
- `npm run server` - Starts the Express API server.
- `npm run build` - Builds the project for production.

## 🛠 Tech Stack

- **Frontend:** TypeScript, Vite, Vanilla CSS.
- **Backend:** Node.js, Express.
- **Database:** Supabase (PostgreSQL).
- **AI Integration:** OpenAI / DeepSeek (via API).
