# Frontend (src/client/)

This directory contains the user interface and client-side logic.

## 📄 Key Files

- `main.ts` - Main entry point for the frontend, handles screen navigation, auth UI, and game rendering.
- `style.css` - Global styles, variables, and animations.

## 🖥 UI Flow

- **Main Menu (`#screen-menu`)** - Starting point after login.
- **Wizard (`#screen-wizard`)** - World creation and character selection.
- **Forge (`#screen-forge`)** - Character and skill creation.
- **Vault (`#screen-vault`)** - Character collection.
- **Game UI (`#game-ui-screen`)** - The main 4-panel gameplay interface.

---
*AI Tip: Modify `main.ts` for UI logic change and `style.css` for visual updates.*
